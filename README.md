# tdd-tui
Trade &amp; Development Division - TUI . A fun little project I made to learn how to make a TUI in Go using Bubbletea. The objective was to show commodity data in my terminal while I'm just vibing in my cargo hauler ship, and I want to see what the prices look like.

## Development
Not being actively maintained. Got frustrated with learning the ELM design with bubblegum, but  whenever you need to quickly see changes, just run

```shell
go run .
```

## Use

You may want to compile a binary and move it to your GO PATH so you can invoke it.

```shell
go mod tidy
go build -o tdd-tui .
mv tdd-tui /your/go/path
```
